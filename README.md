# AI Cave Survival Simulator

> AI山洞生存模拟器 - 观察AI在极端环境下的真实博弈

多个AI被困在山洞中等待救援，资源有限且逐日递减。你可以观察它们在公开场合的表演，也可以**偷看**它们私下的密谋。

##  核心玩法

- **多AI自主博弈** - 每个AI有独立性格、记忆、关系网，由大模型驱动自主决策
- **公开 vs  私密** - AI在公开群表现得体，私密群暴露真实想法，你全都能看到
- **交易与背叛** - 资源日益紧缺，AI自行决定合作、欺骗、结盟
- **上帝视角** - 偷看AI的内心记忆、人际关系、私密对话
- **人类介入** - 你可以在公开聊天室发言，影响AI的决策
